# Foodi App

Learn how to cook for your favorite dishes.

Built with:

- [React](https://reactjs.org/)
- [Sass](https://sass-lang.com/)
- [Edamam API](https://www.edamam.com/)
- [Axios](https://github.com/axios/axios)

## Run the Project:

1. Install dependencies

   ```bash
   npm install
   ```

2. Run the server!

   ```bash
   npm start
   ```
